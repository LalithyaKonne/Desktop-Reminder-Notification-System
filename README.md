# Desktop Reminder Notification System

## Overview

Desktop Reminder Notification System is a Python-based desktop application that helps users schedule reminders with voice notifications and desktop alerts. The application provides a user-friendly graphical interface for creating and managing reminders.

## Features

- Create and schedule reminders
- Desktop notifications using Plyer
- Voice notifications using Google Text-to-Speech (gTTS)
- Calendar-based date selection
- Multiple reminder support
- Custom reminder icons
- Notification history management
- SQLite database integration for storing reminder history
- Interactive GUI built with Tkinter

## Technologies Used

- Python
- Tkinter
- SQLite
- Plyer
- gTTS
- Pygame
- tkcalendar
- Pillow (PIL)

## Installation

### Clone the Repository

```bash
git clone https://github.com/LalithyaKonne/Desktop-Reminder-Notification-System.git
```

### Navigate to Project Directory

```bash
cd Desktop-Reminder-Notification-System
```

### Install Required Packages

```bash
pip install pillow
pip install plyer
pip install pygame
pip install gtts
pip install tkcalendar
```

## Run the Application

```bash
python notify.py
```
